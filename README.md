PORTFOLIO:

A sleek, responsive, and modern portfolio website built to showcase my journey as a full-stack developer. This project highlights my technical skills, featured projects, and professional background with a focus on clean UI and optimal performance.

Live Demo: anoopportfolio-nine.vercel.app

🚀 Features:
Responsive Design: Optimized for all screen sizes, from mobile devices to large desktops.

    Project Showcase: Detailed cards for major projects like Scenic Wonders and a Social Impact Blood Donation Platform.

    Interactive UI: Smooth transitions and modern components built with Tailwind CSS.

    Optimized Performance: Fast load times and high Lighthouse scores via React.

    Contact Integration: Functional contact section for professional inquiries.

🛠️ Tech Stack

    Frontend: React.js

    Styling: Tailwind CSS

    Deployment: Vercel

    Version Control: Git & GitHub

🛠️ Installation & Setup

If you want to run this project locally, follow these steps:

    Clone the repository:
    Bash

    git clone https://github.com/elvyn0/portfolio.git

    Navigate to the project directory:
    Bash

    cd portfolio-repo-name

    Install dependencies:
    Bash

    npm install

4. **Start the development server:**
   ```bash
   npm start
   ```
